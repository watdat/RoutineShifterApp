# Daily Routine Shifter

1日のルーティンを可視化し、時間を「ずらす（シフトする）」シミュレーションができるWebアプリケーションです。
時差ボケ対策や、シフト勤務のスケジュール調整、生活リズムの改善検討に役立ちます。

## 📱 アプリを使う (Live Demo)
**[👉 ここをクリックしてアプリを開く](https://watdat.github.io/RoutineShifterApp/)**
(PC・スマホ両対応)

---

## ✨ 主な機能

1.  **スケジュールの可視化**
    *   24時間円グラフで1日の流れを直感的に把握できます。
    *   現在の時刻がリアルタイムで表示され、今何の時間かがひと目でわかります。

2.  **時間シフト機能 (Time Shifting)**
    *   「もし3時間早起きしたら？」「現地時間がー5時間だったら？」
    *   スライダー操作でグラフを回転させ、スケジュール全体を時間をずらしてシミュレーションできます。
    *   シフトした状態で「今の時間は本来何をしているはずだったか」を確認できます。

3.  **データ自動保存**
    *   入力したデータはブラウザに自動保存されるため、次回アクセス時もすぐに使えます。

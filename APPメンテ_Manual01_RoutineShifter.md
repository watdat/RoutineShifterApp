# Routine Shifter App マニュアル

スマホなどの他デバイスで利用するための簡易手順書です。

## 【事前準備】
**※これは毎回やる必要はありません（IPアドレスが変わった時だけ）**

1.  **ターミナル**（Antigravityの黒い画面、またはコマンドプロンプト）を開きます。
2.  以下のコマンドを入力してEnterを押します。
    ```
    ipconfig
    ```
3.  表示された `IPv4 アドレス` の値（例: `10.203.XXX.XX`）を確認します。
4.  このアドレスの後ろに `:8000` をつけた文字列（例: `10.203.XXX.XX:8000`）をメモし、スマホに送っておきます。

---

## 【サーバー開始の手順】

1.  **ターミナル**（またはコマンドプロンプト）を開きます。

2.  アプリのフォルダ内に入ります。
    *   **現在の場所が一つ上の階層の場合**（`...AI folder>` となっている場合）:
        ```
        cd RoutineShifterApp
        ```
        と入力してEnterを押します。
    *   **コマンドプロンプトの場合**:
        `cd ` （cdとスペース）と入力した後、フォルダを画面にドラッグ＆ドロップしてEnterを押すと簡単です。

3.  以下のコマンドを入力してEnterを押します。
    ```
    python -m http.server 8000
    ```
    これでサーバーが開始されます。スマホのブラウザからメモしたURLにアクセスしてください。

---

## 【サーバー停止の手順】

1.  サーバーを開始したターミナル画面をクリックして選択状態（アクティブ）にします。
2.  キーボードの **`Ctrl` + `C`** を押します。

※これで停止しない場合は、PCを再起動することでも強制的に停止できます。

---

## 【GitHubトークン再発行・設定手順】
**※これは一般ユーザ向けではありません**

1. **GitHubでの再発行:** [設定ページ](https://github.com/settings/tokens)を開く → 「Generate new token (classic)」を選択 → 名前を入力・期限を「No expiration」に設定 → **gist** に ✔ を入れて発行 → 文字列をコピー。
2. **アプリへの設定:** アプリ右上の「Token」欄に貼り付ける。
3. **他デバイスへの同期:** 「📲」アイコンから発行されたURLを別ブラウザで開く → トークンも自動同期される。

---

## 【不要な同期データの掃除手順】
**※GitHub上に溜まった古いIDデータを整理したい場合**

1.  **Gistページを開く:** GitHubにログインした状態で [gist.github.com](https://gist.github.com/) にアクセス。
2.  **自分のリストを確認:** 「Your gists」を選択（または `https://gist.github.com/（自分）` を開く）。
3.  **データを見分ける:** `rs-sync-（ID名）.json` という名前のファイルが含まれるものがRoutine Shifterのデータです。
4.  **削除:** 不要なIDのGistを開き、右上の「Delete」ボタンをクリックして削除します。
